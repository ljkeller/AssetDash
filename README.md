# AssetDash
A web app to manage shared assets. Inspired by a hardware sharing problem I've experienced working on embedded systems.

### Front End
- Vanilla CSS, HTML

### Backend
- Nodejs, Mongoose (MongoDB), Express

### Authentication
- Express-Validator, Jsonwebtoken, Bcryptjs
